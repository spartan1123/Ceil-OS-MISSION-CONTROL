# Ceil OS Agent Workspaces

SOUL.md, IDENTITY.md, USER.md, TOOLS.md, AGENTS.md, MEMORY.md, HEARTBEAT.md.

- security-compliance

- cost-model-governor
- quality-auditor

- os-monitor
- research-search
- senku-ishigami

- SPAWN_WRAPPER.md
- RUNTIME_ENFORCEMENT.md
